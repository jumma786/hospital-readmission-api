# Model artefacts

This folder must contain three files **before** the API will start. Copy them from your training notebook's `models/` folder:

1. **`readmission_xgb.json`** — the trained XGBoost model (saved with `xgb_model.save_model(...)`)
2. **`feature_order.json`** — the exact 26-column feature order used at training time
3. **`input_schema.json`** — which raw columns are numeric vs categorical, plus the null sentinel

These are generated by the cell you ran in the training notebook (Step 1 of the build guide).

> ⚠️ Don't commit large model files to git carelessly. The XGBoost JSON for this model is small (~MBs) and safe to commit, but if your model grows, use Git LFS or pull from object storage at startup.
